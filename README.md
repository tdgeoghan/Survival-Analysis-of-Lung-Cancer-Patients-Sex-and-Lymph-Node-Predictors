# Survival-Analysis-of-Lung-Cancer-Patients-Sex-and-Lymph-Node-Predictors
Biostatistical analysis of Haberman’s survival dataset using R, exploring relationships between clinical variables (age, lymph node status, and operation year) and survival outcomes through exploratory visualization and Kaplan–Meier methods.
This project analyzes the Haberman’s Survival Dataset to explore relationships between clinical variables and patient survival outcomes. The goal is to apply exploratory data analysis and survival analysis techniques commonly used in biostatistics and public health research.

## Dataset
The dataset includes:
- Age of patient at time of surgery
- Year of operation
- Number of positive axillary lymph nodes
- Survival status (survived or did not survive)

## Methods
The analysis was conducted in R using the following methods:

- Data cleaning and variable recoding
- Exploratory data analysis using ggplot2
- Comparison of lymph node distribution across survival groups
- Kaplan–Meier survival analysis using the survival and survminer packages

## Key Visualizations
- Age distribution by survival status
- Boxplots of lymph node involvement by survival outcome
- Kaplan–Meier survival curves

## Tools Used
- R
- ggplot2
- survival
- survminer

## Notes
This dataset is used for educational purposes and does not represent modern clinical lung cancer data. The survival analysis demonstrates foundational biostatistical techniques rather than clinical prediction.
